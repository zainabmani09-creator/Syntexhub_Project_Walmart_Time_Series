# Syntexhub Project – Walmart Sales Analysis

## Project Overview
This project analyzes Walmart sales data using time series and category-based visualizations.
It is part of Week 2 Project under the Syntexhub learning program.

## Objectives
- Perform time series analysis on weekly sales data
- Compare sales across store categories
- Visualize sales share using pie charts
- Save all charts as PNG files
- Document chart choice and aggregation

## Dataset
Walmart Sales Forecasting Dataset

Files used:
- train.csv
- stores.csv

## Time Series Aggregation
The dataset provides weekly sales data.
Therefore, time series analysis was performed using weekly aggregation.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations
- Walmart weekly sales trend over time (line chart)
- Sales comparison by store type (bar chart)
- Store type sales share (pie chart)

## Summary
The analysis highlights sales trends and store-type contributions,
demonstrating the importance of data visualization in retail analytics.
